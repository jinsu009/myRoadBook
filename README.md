# 독서기록 앱 (Book-Log-App)

바코드 스캔을 통한 나만의 독서 기록 관리 

## 개발 환경 및 도구 (Development Environment)

### ***Core Stack***

- **Framework** : React Native (Expo SDK 50+) (expo@55.0.7)
- **Language** : TypeScript (v.)
- **Runtime** : Node.js (v24.13.0)
- **Package Manage : npm**

### **Development Tools**

- **Camera/Barcode** : “expo-camera” 또는 “expo-barcode-scanner”
- **Data Fetch : “Axios” 또는 “Fetch API”**
- **Storage** : “@react-native-async-storage/async-storage” .. 로컬 저장용
- **Icons** : “@expo/vector-icons”
